NCR Ventures 3 Opportunities to Be Better Week Ending 4.26 — Patios Filling, Volume Climbing
5 of 6
1
The Check-Back Is Still the Soft Spot
A guest told us the server only asked how the food was "before we had even started eating." Another said the server offered to-go boxes for food the guest clearly didn't finish, and never asked if anything was wrong. A birthday party with no drink check-back, no side plates, no manager touch. Two-bite check-back. Every table. Every shift.
2
Plate-Up Discipline at the Pass
Multiple food complaints this week all point to the same place — the pass. Pasta sauced too heavy. Chicken cooked dry. Pizzas soggy from over-saucing or sitting too long. An entree to the wrong guest because the server didn't write it down. Right ticket, right plate, right window time, right hand-off. If something looks wrong, we don't run it. We fix it.
3
Special Occasion Flags Are Slipping Through
A guest booked a birthday dinner, marked the occasion on the reservation, and wasn't acknowledged. Second week in a row a guest has flagged this. Pre-shift: pull every occasion off the res sheet. Read it out loud. Assign one person to own the moment. Every flag gets touched.
